# Welcome to the CASim Web UI Component Library

This library provides a collection of reusable web UI components designed to streamline the development of user interfaces for CASim applications. Whether you're building dashboards, data visualizations, or interactive tools, these components will help you create consistent and efficient UIs.

## Getting Started

To get started with the CASim Web UI Component Library, follow these steps:

1. **Installation**: download the library from the official repository [casWOW](https://github.com/ComponentAssemblySystems/casWOW) and include it in your project.
    - you will want to take all of the `.css` files from the `docs/css/` folder and add them to your existing `css` folder.
    - you will also want to take all of the `.js` files from the `docs/js/` folder and add them to your existing `js` folder.
2. **Stories**: Explore the stories in this Storybook to see examples of how to use each component.
    - Each component has its own story that demonstrates its usage, properties, and variations.
    - These can be found in the project directory under `src/stories/`.
3. **Documentation**: Refer to the documentation provided in each story for detailed information on components, including props, methods, and events.

For more detailed instructions on installation and setup, please refer to the [Getting Started](/docs/getting-started--documentation) guide.

## Code Examples

Code examples for each component can be found within their respective stories. These are starter templates to help you understand how to implement and customize the components for your specific needs.

If you encounter any issues or have questions about using the CASim Web UI Component Library, please reach out to the support team via the [casWOW GitHub repository](https://github.com/ComponentAssemblySystems/casWOW/issues/new).
