# Definition

Navigation is a set of interactive elements that allow users to move through different sections of an application or website.

## Usage

This component is placed at the top of the page and should contain the most important links for the site. It can also include user actions such as login/logout.

### Variants

The navigation bar can adapt based on the user's display size (responsive design) and their access level.
