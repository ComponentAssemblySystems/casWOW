# Colors

The CAS color palette is designed to provide a cohesive and accessible visual experience across all platforms. It includes a range of colors for various use cases, including primary, secondary, accent, and feedback colors.

## Themes

By default, CAS supports two themes: Light and Dark. Each theme has its own set of color variables to ensure optimal readability and aesthetics in different lighting conditions. The colors are defined using hexadecimal color codes, unless otherwise specified.
