# Variables

The CAS WOW design system includes a comprehensive set of design tokens that define the visual properties of the UI components. These tokens ensure consistency across the application and make it easier to maintain and update the design.

## Sizing

The sizing variables define standard dimensions for spacing, widths, heights, and other size-related properties. These variables help maintain a consistent layout and spacing throughout the application.

All sizing variables are defined using `rem` units to ensure scalability and responsiveness across different screen sizes and user settings. The base font size is typically set to `16px`, so `1rem = 16px`. Here are the sizing variables used in the CAS WOW design system:

```css
--cas-spacer-xxs: 0.25rem;    /* 4px */
--cas-spacer-xs: 0.5rem;      /* 8px */
--cas-spacer-sm: 0.75rem;     /* 12px */
--cas-spacer: 1rem;           /* 16px */
--cas-spacer-md: 1rem;        /* 16px */
--cas-spacer-lg: 1.25rem;     /* 20px */
--cas-spacer-xl: 1.5rem;      /* 24px */
--cas-spacer-xxl: 2rem;       /* 32px */
--cas-spacer-xxxl: 3rem;      /* 48px */
```

## Typography

### Font Family

```css
--cas-font-serif: "IBM Plex Serif";
--cas-font-family-sans-serif: "IBM Plex Sans";
--cas-font-family-monospace: "IBM Plex Mono";
--cas-display-font-family: "Inter Display";
--cas-font-family-fa: "Font Awesome 5 Free";
--cas-font-family-code: "IBM Plex Mono";
```

### Font Sizes

```css
--cas-font-size-root: null;
/* 12px */
--cas-font-size-xs: calc(var(--cas-font-size-base) * .75);
/* 14px */
--cas-font-size-sm: calc(var(--cas-font-size-base) * .875);
/* 16px */
--cas-font-size-base: 1rem;
/* 20px */
--cas-font-size-lg: calc(var(--cas-font-size-base) * 1.25);

/* 48px */
--cas-font-size-h1: 3rem;
/* 32px */
--cas-font-size-h2: 2rem;
/* 28px */
--cas-font-size-h3: 1.5rem;
/* 24px */
--cas-font-size-h4: 1.25rem;
/* 20px */
--cas-font-size-h5: 1.125rem;
/* 16px */
--cas-font-size-h6: var(--cas-font-size-base);
```

### Font weights

--cas-font-weight-medium: 500;
--cas-font-weight-semibold: 600;
--cas-font-weight-bold: 700;
--cas-font-weight-bolder: 900;
```

```
