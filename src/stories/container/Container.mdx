# Definition

The Container component is a layout wrapper that centers and constrains the width of the content within it. It provides responsive behavior by adjusting its width based on predefined breakpoints, ensuring optimal readability and presentation across various screen sizes.

## Usage

The Container component is used to wrap page content, providing a consistent and centered layout. It can be configured to have a fluid width that spans the entire viewport or a fixed width that adjusts at specific breakpoints for better content alignment.

## Resources

[Bootstrap documentation / Container](https://getbootstrap.com/docs/5.3/layout/containers/)

## Default implementation

The default container class adjusts to the screen size, filling the available space of the parent wrapper.

## Additional variations

To change the container width, update the `container` wrapper to include the breakpoint sizing, such as `-sm`, `-md`, `-lg`, `xl`, or `xxl`.

For a full width container that spans the entire width of the viewport, use the `container-fluid` class.
